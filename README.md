## BetArena – Fictional Sports Betting App

**BetArena** is the imaginary context used across the coding exercises in this repository. Candidates are presented with real-world-style problems drawn from its domain so that technical skills can be evaluated in a meaningful, practical setting.

### About BetArena

BetArena is a fictional online sports betting platform where registered users can browse live and upcoming sporting events, review odds, and place bets using a virtual wallet balance. The platform supports multiple sports and markets, provides real-time updates, and keeps a full history of every transaction.

---

### Main Features

#### 1. User Accounts
- Registration and login with email/password or OAuth providers (e.g. Google).
- Profile management: personal details, notification preferences, and responsible-gambling limits.
- KYC (Know Your Customer) verification flow for account activation.

#### 2. Virtual Wallet
- Each user has a wallet with a virtual balance (no real money involved).
- Deposit and withdrawal flows (simulated payment gateway).
- Full transaction history with timestamps and status.

#### 3. Sports & Events Catalogue
- Browse a catalogue of sports (football, basketball, tennis, etc.).
- View upcoming and live events with real-time status updates.
- Search and filter events by sport, league, date, or keyword.

#### 4. Odds & Markets
- Each event exposes multiple betting markets (e.g. match result, over/under, first scorer).
- Odds update in real time as events progress or market conditions change.
- Support for decimal, fractional, and American odds formats.

#### 5. Bet Placement
- Single bets and accumulator (parlay) bets.
- Stake validation against wallet balance and configurable bet limits.
- Bet confirmation with a summary of potential payout before submission.

#### 6. Bet History & Settlement
- Full list of open, settled, and cancelled bets per user.
- Automatic settlement when event results are published.
- Push and email notifications for bet outcomes.

#### 7. Responsible Gambling Controls
- Self-imposed deposit limits, loss limits, and session time limits.
- Cooling-off periods and self-exclusion options.
- Flagging of unusual betting patterns for review.

#### 8. Admin Back-office
- Manage sports, events, and markets (create, update, suspend, close).
- Monitor user activity and wallet movements.
- Override bet settlement and handle dispute resolution.

### Tech Interview Usage

Exercises in this repository will progressively introduce features of BetArena as the context for coding challenges, system-design discussions, and code-review sessions. Pull requests will be opened for candidates to review, extend, or refactor.

Feel free to ask questions about the domain during your interview — understanding the problem space is part of the exercise.
